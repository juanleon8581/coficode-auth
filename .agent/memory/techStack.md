# Tech Stack

Below is the chosen technical stack for the project and the accompanying tools.

## Core Framework

- **Next.js** (v16.1.6): React framework for full rendering and routing.
- **React** (v19.2.3): UI library.

## Styling and UI Components

- **Tailwind CSS** (v4): Utility-first CSS framework for agile design.
- **Shadcn UI**: Collection of highly customizable UI components, using libraries like `radix-ui`, `class-variance-authority`, `clsx`, and `tailwind-merge`.

## Authentication and Database

- **Supabase Auth**: Complete authentication service (initial implementation in progress).

## Form Handling and Validation

- **React Hook Form**: Agile and efficient state and error management in forms.
- **Zod**: Schema declaration and input/type validation.

## Language Tools and Linters

- **TypeScript** (v5): Typed superset for JavaScript.
- **ESLint** (v9): Linter and static analysis to maintain good practices and clean code.

## Testing

- **Vitest**: Fast execution unit and integration testing framework.
- **React Testing Library**: Utilities to test React components from the user's perspective.
